Impressão pelo Sistema

Usa o diálogo de impressão nativo do Android. Funciona com:

  • Impressoras Bluetooth pareadas no celular
  • Impressoras USB conectadas via cabo OTG
  • Google Cloud Print / Save as PDF
Para impressoras térmicas Bematech, Wi-Fi é mais confiável e imprime sem diálogos.
